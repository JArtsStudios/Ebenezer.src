Política de Privacidad

Última actualización: 26/01/2026

Sitio: ebenezersrc.org Contacto: j.arts.studios.01@gmail.com

Esta Política de Privacidad describe cómo se recopila, usa y protege la información cuando visitas y utilizas el sitio y las páginas relacionadas con el contenido multimedia de Ebenezer SRC.

1. Qué información recopilamos

  • Datos técnicos básicos: como navegador, tipo de dispositivo y registros de errores (solo para funcionamiento).
  • Datos de YouTube: el sitio consulta la API de YouTube para mostrar videos, estadísticas públicas y comentarios públicos.
  • Inicio de sesión con Google (opcional): si decides iniciar sesión, se usa OAuth de Google para obtener autorización.

2. Cómo usamos la información

  • Para mostrar contenido (videos, transmisiones, podcasts) desde YouTube.
  • Para mejorar el funcionamiento del sitio y corregir errores.
  • Si inicias sesión, para permitir acciones como dar “like” o comentar (cuando esté habilitado).

3. Datos compartidos con terceros

El sitio utiliza servicios de terceros (YouTube/Google) para la reproducción y obtención de datos. La información puede ser procesada por Google según sus políticas.

4. Cookies

YouTube y Google pueden usar cookies o tecnologías similares para reproducir contenido y gestionar sesiones. Nosotros no usamos cookies con fines publicitarios propios.

5. Conservación y seguridad

No almacenamos contraseñas. Si se usan tokens de acceso, se manejan según la configuración de OAuth y solo para la sesión necesaria. Recomendamos usar HTTPS siempre.

6. Tus derechos

Puedes solicitar información o eliminación de datos relacionados (si aplica) escribiendo a: j.arts.studios.01@gmail.com.

7. Contacto

Si tienes preguntas sobre esta Política de Privacidad, contáctanos: j.arts.studios.01@gmail.com
